# 🎙️ ESP32 Voice Recognition System

## 📌 Overview

This project implements an **offline voice recognition system using ESP32**, enabling users to control devices through voice commands without requiring an internet connection. It combines Bluetooth communication, real-time processing, and OLED display output for an efficient embedded system solution.

---

## 🚀 Features

* 🎤 Offline voice recognition (no cloud required)
* 📡 Bluetooth-based command input
* 📟 OLED display for output visualization
* ⚡ Real-time response system
* 🔌 Low-cost and efficient embedded design

---

## 🛠️ Technologies Used

* ESP32 Microcontroller
* Arduino IDE
* Embedded C/C++
* Bluetooth Serial Communication
* I2C Communication (OLED SSD1306)

---

## 🔌 Hardware Requirements

* ESP32 Development Board
* OLED Display (SSD1306)
* Microphone / Voice Recognition Module
* Jumper Wires
* Power Supply

---

## 🧠 Working Principle

1. Voice command is captured through the microphone/module
2. Command is processed and matched with predefined instructions
3. ESP32 executes the corresponding action
4. Output is displayed on the OLED screen

## ⚙️ Installation & Setup

1. Install Arduino IDE
2. Install ESP32 board support
3. Install required libraries:

   * WiFi.h
   * BluetoothSerial.h
   * Adafruit_GFX
   * Adafruit_SSD1306
4. Upload the `.ino` file to ESP32
5. Connect hardware as per circuit diagram
